Какой алгоритм деплоя нам нужен, чтобы было всё хорошо


При новом коммите в репозитории blog-generator:
    0. git clone (git pull)
    1. npm i
    2. npm run build
    3. npm run export
    4. переместиться в другую папку и сделать git clone (git pull) justaddacid.github.io
    5. удалить директории:
        _next
        assets
        favicon
        posts
    6. переместить файлы из blog-generator/out в корень justaddacid.github.io (с заменой)
    7. git add -A
    8. git commit -m "ahaha benis"
    9. git push origin master
